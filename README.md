This repository contains the data processing and analysis pipeline for a randomized, double-blind, placebo-controlled study investigating the effects of N-acetylcysteine (NAC) on approach–avoidance biases and craving in cocaine-dependent patients undergoing inpatient detoxification.

#Study Design. 
    Summary Population: Cocaine-dependent adults (18–65 years) 
    Setting: Addiction Medicine Unit 73, Brugmann University Hospital, Brussels 
    Design: Randomized, double-blind, placebo-controlled 
    Intervention: 
        NAC group: 2,400 mg/day oral N-acetylcysteine 
        Placebo group: matching capsules 
    Duration: 5 consecutive days 
    Primary outcomes: Approach–avoidance bias (R-SRC score) Craving (CCQ-Brief) 
    Secondary outcomes: Mood, anxiety, impulsivity, metacognition, hedonic capacity

#Data Description. 
    R-SRC / Approach–Avoidance Task Stimuli: 16 cocaine-related images 16 neutral images 
    Blocks: Compatible (approach cocaine / avoid neutral) Incompatible (avoid cocaine / approach neutral) 
    Trials: 64 trials per block (128 total)
